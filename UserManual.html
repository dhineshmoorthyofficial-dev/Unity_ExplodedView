Exploded View Tool User Manual

Welcome! This guide will help you get started with the Exploded View Tool for Unity. This package works like magic to pull your 3D models apart for cool visualizations.

What does this package do?

  • Spherical Explosion: Blasts parts outwards from the center (like an actual explosion).
  • Target Mode: Lets you precisely place where each part should go for technical diagrams.
  • Safe & Reversible: It never breaks your original model. You can always reset.

How to Use the Modes

1. Spherical Mode (Default)

Best for: Fun effects, sci-fi holograms, quick breakdowns.

Parts fly away from the center. You can adjust the Sensitivity to make them fly further.

2. Target Mode (Precision)

Best for: Instruction manuals, showcasing specific parts.

1. Switch Explosion Mode to Target.
2. Click Create Targets.
3. You will see "Ghost" objects appear. Move these ghosts to where you want the parts to land.
4. The slider now animates parts perfectly to your custom positions.

3. Curved Mode (Advanced)

Best for: Cinematic reveals, organic movements, complex paths.

1. Switch Explosion Mode to Curved.
2. The tool creates a path with 2 default control points (at 5% and 10% of the path).
3. Drag these points in the scene or add more points in the Control Points list to warp the path.
4. Each part follows its own unique spline path!

Common Questions

Q: I moved my object and the explosion looks weird now.
A: Just click Reset & Setup Explosion again to refresh the tool with the new positions.

Q: How do I delete it safely?
A: Look for the red "Danger Zone" at the bottom of the component and click Deep Remove. This cleans up all the hidden helper objects for you.
